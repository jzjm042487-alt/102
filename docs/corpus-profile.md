因为纯切利用率达不到目标(常见 99.25%), 需靠焊接吃掉料头把利用率顶上去。
因此分类器不能只看几何, 真正判据是 **纯切能否达标目标利用率**:

- 纯切可达标 → 纯切快路(0 焊口, 跳过焊接建模)。
- 纯切不达标 / 几何不可纯切 → 焊接 arc-flow(coarse-to-fine 网格)。
